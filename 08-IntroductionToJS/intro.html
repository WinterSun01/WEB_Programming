Вывод на экран

  1. Модальные окна:
    • alert(); //отображение модального окна сообщения (как в MessageBox())
    • prompt(); //модальное окно со строкой ввода. Функция возвращает введённую строку
    • confirm(); //модальное окно "Да", "Нет"
  2. Вывод в консоль браузера: 
    console.log(.......)
  3. document.write();//выводит данные непосредственно на страницу; DEPRECATED
  4. Вывод на страницу при помощи DOM - Document Object Model, методами InnerHTML(), OuterHTML()....

Строки

  1. "string";

Переменные и типы данных

  1. var - объявляет переменную, создаёт глобальную переменную, независимости от того, в какой области видимости она объявлена;
  2. let - объявляет переменную, переменные созданные при помощи let существуют только в той области видимости, в которой они объявлены;
  3. const - объявляет константу, т.е. константы существуют только в той области видимости, в которой они объявлены;
